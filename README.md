# Controlled Chaos

## About:
This application was created to help manages tasks in the users every day life.
The utilization of sticky notes allows you to keep reminders.

## Utilization:

* Javascript
* HTML5
* CSS
* MongoDB
* Express
* React

## Setup:

To launch frontend application
```
cd client
npm install
npm start
```

To launch backend application

```
cd server
npm install
nodemon server.js
```
